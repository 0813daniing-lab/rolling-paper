# 로그인/회원가입 입력 항목 간소화 + 직급 추가

관리자 로그인과 회원가입 화면을 이메일/전화번호 없이 사용할 수 있도록 수정하고, 직급 입력을 추가했습니다.

- 로그인: 아이디, 비밀번호
- 회원가입: 담당 트랙, 기수, 이름, 직급, 아이디, 비밀번호
- 내 정보: 담당 트랙, 기수, 이름, 직급

Supabase Auth는 내부적으로 이메일 형식이 필요하므로, 입력한 아이디를 내부 인증용 이메일로 자동 변환해 저장합니다. 화면에는 이메일/전화번호를 노출하지 않습니다.

직급 저장을 위해 Supabase SQL Editor에서 `supabase/profile-position.sql`을 한 번 실행해야 합니다.
